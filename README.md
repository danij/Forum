# Fast Forum Backend

Backend software for providing a discussion forum. 

## Goals

* Fast response times
* Memory efficiency
* Avoiding unnecessary code
* Cross-platform

## Coding Style

* 4 SPACES for indent
* camelCase
* Brackets on new line (C++)

## Building

    mkdir build
    cd build
    cmake ../
    make

## Running Tests

After build, from the build folder:

    ./test/ForumServiceTests/ForumServiceTests
